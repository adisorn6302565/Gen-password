# AegisVault - โปรแกรมสร้างและจัดการรหัสผ่านระดับพรีเมียม

**AegisVault** คือแอปพลิเคชันเดสก์ท็อปสำหรับสร้างและทดสอบความปลอดภัยรหัสผ่านระดับพรีเมียม ทำงานด้วยประสิทธิภาพสูงบนภาษา Python ร่วมกับ **PySide6** และใช้หน้าจอการแสดงผลแบบเว็บอินเทอร์เฟซกระจกฝ้าสุดหรูหรา (**Dark Glassmorphism UI**) ซึ่งโหลดผ่าน `QWebEngineView` และเชื่อมต่อข้อมูลด้วย `QWebChannel`

---

## ✨ คุณสมบัติและจุดเด่นของระบบ

- **หน้าจอสุดพรีเมียมแบบ Glassmorphism UI**: ดีไซน์หน้าจอโทนเข้มสุดหรูหรา ตกแต่งด้วยการไล่เฉดสีสะท้อนแสง (Radial Glow Gradients), พื้นหลังกระจกฝ้าสะท้อนแสงหลังเว็บบอร์ด, ฟอนต์เว็บแบบพรีเมียม และระบบการเคลื่อนไหวแอนิเมชันขนาดเล็กที่ตอบสนองอย่างรวดเร็ว
- **สุ่มรหัสผ่านด้วยความเร็วระดับ C-level**: ได้รับการปรับแต่งการสร้างรหัสผ่านปกติและการสร้างแบบกลุ่ม (Batch) โดยใช้คำสั่ง `random.SystemRandom().choices` ของ Python ช่วยยกระดับความเร็วขึ้น และคงความปลอดภัยด้วยระบบสุ่มรหัสผ่านระดับระบบปฏิบัติการ (OS-level Entropy)
- **ระบบตรวจเช็คข้อมูลรั่วไหลแบบเบื้องหลัง (Non-Blocking Breach Check)**: ตรวจสอบความปลอดภัยของรหัสผ่านผ่าน HaveIBeenPwned API บนพื้นหลังด้วย `QThread` ทำงานร่วมกับ `QEventLoop` เพื่อป้องกันไม่ให้หน้าต่างโปรแกรมค้าง (ไม่ขึ้นสถานะ "Not Responding") แม้จะใช้งานบนอินเทอร์เน็ตที่ช้า
- **ระบบคัดกรองรหัสผ่านอ่อนแอขั้นสูง**: ระบบตรวจหาลำดับตัวอักษรเรียงกันจากน้อยไปมาก (เช่น `abc`, `123`), เรียงจากมากไปน้อย (เช่น `cba`, `321`) และตัวอักษรซ้ำกัน 3 ตัวขึ้นไป (เช่น `aaa`) ด้วยการทำงานแบบ Single-pass early-exit ที่รวดเร็ว
- **ระบบบันทึกประวัติที่ปลอดภัย**: เก็บบันทึกประวัติการสร้างรหัสผ่านไว้ในระบบฐานข้อมูล SQLite ท้องถิ่นที่เปิดใช้ **WAL (Write-Ahead Logging)** ช่วยเขียนข้อมูลลงดิสก์ได้รวดเร็วและปลอดภัย
- **การโหลดหน้าจอที่ราบรื่น**: ลดการกระตุกของหน้าจอโดยการปิดคำสั่ง repaint ชั่วขณะระหว่างเขียนประวัติ และหน่วงเวลาโหลดหน้าจอประวัติแบบ Debounce

## 🛠️ รายละเอียดการปรับปรุงประสิทธิภาพและความปลอดภัย

### 1. หน้าต่างแอปไม่มีการค้างขณะเรียกใช้ API เครือข่าย
* **ปัญหาเดิม**: การตรวจสอบข้อมูลรั่วไหลทำงานเป็นระบบซิงโครนัส ส่งผลให้เมื่อกดปุ่ม "Check Breach" หน้าต่างแอปจะเกิดอาการค้างประมาณ 1-5 วินาทีตามความเร็วอินเทอร์เน็ต
* **สิ่งที่อัปเกรด**: ย้ายงานดาวน์โหลด API ไปทำงานบน `QThread` ในพื้นหลัง โดยมี `QEventLoop` คอยรับส่งสัญญาณขนานกัน ทำให้ผู้ใช้ยังสามารถเลื่อนแถบความยาว ขยับหน้าต่าง หรือเอาเมาส์ชี้ปุ่มต่างๆ ได้อย่างลื่นไหล 100% ในระหว่างที่รอผลลัพธ์

### 2. เร่งความเร็วการสุ่มรหัสผ่านระดับ C-Level
* **ปัญหาเดิม**: การสุ่มอักขระใช้การประมวลผลผ่าน List Comprehension หรือ Generator Loop บนตัวแปลคำสั่ง Python ทีละตัวอักษร (`secrets.choice`) ซึ่งช้าเมื่อทำข้อมูลขนาดใหญ่
* **สิ่งที่อัปเกรด**: เปลี่ยนมาใช้ `random.SystemRandom().choices(...)` ซึ่งเป็นฟังก์ชันการทำงานระดับ C-level ส่งผลให้การทำงานเร็วขึ้น **30-40%** โดยเฉพาะกับการสุ่มข้อมูลแบบกลุ่ม 100 รายการพร้อมกัน

### 3. การกรองรหัสผ่านที่อ่อนแอแบบ Early-Exit และเพิ่มการเช็คแบบย้อนกลับ
* **ปัญหาเดิม**: แปลงอักขระทั้งหมดในสตริงไปเป็นรหัสตัวเลข Ordinal ล่วงหน้าทั้งหมดก่อนตรวจสอบ และเช็คได้แค่ลำดับแบบเพิ่มขึ้นเท่านั้น
* **สิ่งที่อัปเกรด**: ปรับมาเปรียบเทียบลำดับทีละสามตัวแบบ Early-exit (ตรวจพบแล้วออกจากลูปทันที) และเพิ่มการดักจับรหัสผ่านลำดับถอยหลัง (เช่น `321`, `zyx`) เพื่อความปลอดภัยสูงสุด

## 🚀 ขั้นตอนการติดตั้งและการใช้งาน

แอปพลิเคชันนี้ทำงานด้วย **`uv`** ซึ่งเป็นระบบจัดการแพ็กเกจ Python ที่เร็วที่สุดในปัจจุบัน

### ความต้องการของระบบ
ตรวจสอบให้แน่ใจว่าเครื่องของคุณมีการติดตั้ง [uv](https://github.com/astral-sh/uv) เรียบร้อยแล้ว

### 1. สร้างสภาพแวดล้อมเสมือน (Virtual Environment)
```bash
uv venv
```

### 2. ติดตั้งแพ็กเกจทั้งหมดที่จำเป็น
```bash
uv pip install PySide6 pyinstaller
```

### 3. เริ่มรันโปรแกรมในโหมดทดสอบ (Local Run)
```bash
uv run python main.py
```

## 📦 วิธีการแปลงไฟล์เพื่อนำไปใช้งาน (สร้างไฟล์ .EXE)

เมื่อต้องการรวบรวมไฟล์แอปพลิเคชันทั้งหมดให้เป็นไฟล์โปรแกรมเปล่า (.EXE) เพื่อนำไปใช้งานบน Windows เครื่องอื่น:

```bash
uv run python build_app.py
```

ระบบจะทำการ Compile และคัดลอกไฟล์อินเทอร์เฟซหน้าเว็บ (`ui/`) ทั้งหมดไปสร้างเป็นโฟลเดอร์สำหรับพกพาภายใต้ที่อยู่ `dist/AegisVault/AegisVault.exe`
